# Recent OS V0.1

This is the initial version of Recent OS, a simple operating system interface built in Java using JavaFX for the GUI.

## Features
- Basic GUI structure
- JavaFX-based interface
- Controller setup for future enhancements

## Project Structure
- `GUI.java`: Main controller class for the GUI

## How to Run
1. Ensure you have Java and JavaFX installed
2. Compile the Java files
3. Run the application
